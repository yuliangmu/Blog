# Mac 备忘清单

**`⌘`** => **`⊞`** => **`Command`** 

**`⌃`** => **`Ctrl`** => **`Control`**

**`⌥`** => **`Alt`** => **`Option`**

**`⇧`** => **`Shift`** => **`Shift`**

> PC 和 Mac 都使用全称

## Mac OS

- **`Command + C`** + **`Command + Shift + V`** 剪切

- **`Command + ,`** 打开应用设置

- **`Command + Shift + 5`** 屏幕截取

- **`Command + Shift + D`** Toggle 程序坞

- **`Command + Shift + Esc`** 强制退出

- **`Command + Shift + Space`** 打开 Finder

- **`Control + ↑`** Toggle 所有打开的窗口（同`F11`）
