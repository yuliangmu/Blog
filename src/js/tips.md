## 将单个字符串首字母大写

```js
/**
 * @desc: 将单个字符串首字母大写
 * @str {String} - 单个字符串
 * @return: 首字母大写的 str
 */
const capitalizeFistLetter = str => str.charAt(0).toUpperCase() + str.slice(1)
```
